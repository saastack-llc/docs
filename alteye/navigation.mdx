## How navigation works

- The left sidebar shows global pages.
- The server selector scopes actions to a single server.

## Page map

- [Dashboard](/alteye/dashboard-tour): Status, totals, and recent activity.
- [Add a server](/alteye/adding-server): Connect a new Rust server.
- [Kicks](/alteye/managing-kicks): Remove players and review history.
- [Whitelist](/alteye/whitelist-management): Approve access and set expirations.
- [Billing](/alteye/billing): Check subscription status and payment health.
- [Account settings](/alteye/account-settings): Update email, password, or delete account.
- [Access settings](/alteye/server-settings/access): API key and plugin download.
- [Protection settings](/alteye/server-settings/protection): Detection filters and kick message.
- [Notification settings](/alteye/server-settings/notifications): Discord webhook alerts.
- [Billing settings](/alteye/server-settings/billing): Cancel or reactivate a server.

## Quick help

- Review kick history: [Kicks](/alteye/managing-kicks)

- Get API key or plugin: [Access settings](/alteye/server-settings/access)
- Check payment status: [Billing](/alteye/billing)
- Update account info: [Account settings](/alteye/account-settings)
