## What the dashboard shows

- Server cards with status and quick actions
- Recent activity across all servers
- Account totals for kicks and whitelist entries
- Server selector to focus on one server

<Tip>
If you need to manage a player, open the server first to keep actions scoped correctly.
</Tip>

## How to use it

<Steps>
  <Step title="Check status first">
    Review server cards to spot offline or warning states.
  </Step>
  <Step title="Open a server">
    Use the server selector to focus on one server.
  </Step>
  <Step title="Take action">
    Go to Kicks or Whitelist and complete the task.
  </Step>
</Steps>
