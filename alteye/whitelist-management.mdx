## When to use the whitelist

- Run a private or invite-only server.
- Approve access for clan members or event guests.
- Set time-limited access with expiration dates.

## Add a player

<Steps>
  <Step title="Open Whitelist">
    Use the global Whitelist view or a specific server page.
  </Step>
  <Step title="Enter player details">
    Add SteamID, optional reason, and optional expiration.
  </Step>
  <Step title="Save">
    The player can join immediately if the server whitelist is enabled.
  </Step>
</Steps>

## Manage access

- Update or remove entries when access changes.
- Use expirations to end temporary access automatically.
- Keep reasons short and consistent for staff clarity.
