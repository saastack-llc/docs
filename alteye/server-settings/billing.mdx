## What this tab controls

- Payment status and next renewal date
- Cancel or reactivate a server subscription

## Open the Billing tab

<Steps>
  <Step title="Open the server">
    Select the server from the dashboard.
  </Step>
  <Step title="Go to Server Settings">
    Open the Billing tab for this server.
  </Step>
</Steps>

## Review status

Check the current status and the next renewal or cancellation date.

## Cancel or reactivate

- Cancel in the Billing tab if you no longer need that server.
- Reactivate from the same tab if a subscription was canceled.

<Tip>
If a server shows Payment Required, update billing before troubleshooting connection issues.
</Tip>
