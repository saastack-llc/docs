## What this tab controls

- API key used to authenticate server requests
- Plugin download for Oxide or Carbon

## Open the Access tab

<Steps>
  <Step title="Open the server">
    Select the server from the dashboard.
  </Step>
  <Step title="Go to Server Settings">
    Open the Access tab.
  </Step>
</Steps>

## API key

- Copy the key when you need to update server config.
- Regenerate the key if it was exposed.

<Warning>
Regenerating the key invalidates the old key immediately.
</Warning>

## Download the plugin

- Choose your server software (Oxide or Carbon).
- Click Download Plugin and install it on your server.
