## What this tab controls

- Discord webhook for detection alerts
- Enable or disable notifications per server

## Open the Notifications tab

<Steps>
  <Step title="Open the server">
    Select the server from the dashboard.
  </Step>
  <Step title="Go to Server Settings">
    Open the Notifications tab.
  </Step>
</Steps>

## Add or update the webhook

- Paste the Discord webhook URL.
- Click Save Webhook to confirm.
- Clear the field to disable notifications.

<Note>
AltEye sends a test message before saving a new webhook.
</Note>

<Tip>
Treat the webhook URL like a password and keep it private.
</Tip>
