## What this tab controls

- Detection filters that block suspicious connections
- The kick message shown by the anti-cheat system

## Open the Protection tab

<Steps>
  <Step title="Open the server">
    Select the server from the dashboard.
  </Step>
  <Step title="Go to Server Settings">
    Open the Protection tab.
  </Step>
</Steps>

## Detection filters

- **Tor**: Recommended, very low false positives.
- **VPNs**: Recommended, reliable signal.
- **Proxies**: Recommended, good backup signal.
- **Datacenter IPs**: Caution, can block some legitimate networks.
- **Heuristic Detection**: Caution, broader checks may false-flag.
- **Abuse / Bad Reputation**: Risky, reputation data can be noisy.

<Tip>
Start with the recommended filters and add the cautious ones only if needed.
</Tip>

## Kick message

Update the message shown to players who are kicked by the anti-cheat system, then save.
