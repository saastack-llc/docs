## Before you start

- Server IP address (IPv4)
- Server software: Oxide or Carbon
- Access to in-game chat or server console

## Add the server

<Steps>
  <Step title="Open Add Server">
    From the dashboard, click Add Server.
  </Step>
  <Step title="Enter server IP">
    Add your game server IPv4 address.
  </Step>
  <Step title="Select server software">
    Choose Oxide or Carbon for the plugin download.
  </Step>
</Steps>

<Note>
Only IPv4 addresses are accepted in the add server form.
</Note>

## Complete subscription (if prompted)

If payment is required, finish the subscription checkout to activate the server.

## Download the plugin

After checkout, download the plugin for your server software and install it:

- Oxide: `oxide/plugins`
- Carbon: `carbon/plugins`

## Set the API key

Once the plugin is installed, set the API key shown in the modal:

- In-game chat: `/alteyekey <api_key>`
- Console: `alteyekey <api_key>`

<Warning>
Keep your API key private. Treat it like a password.
</Warning>
