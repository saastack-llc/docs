## What AltEye does

- Manage one or more Rust servers from a single dashboard.
- Kick players, track reasons, and review history.
- Control whitelist access per server.
- Send real-time alerts with Discord webhooks.

## What you need

- An AltEye account
- Your server IP address
- Server software: Oxide or Carbon
